HTML är förkortning av “Hyper Text Markup Language.” På engelska detta kallas som (language/Språk) det är inte en riktig programmering språk. Man kan se detta som en texteditor som man skriver websidans grunder. CSS är förkortning av “Cascading Style Sheet.” som namnet också avslöjar; det är ett verktyg som vi stylar Webbsidorna. CSS är ett kraftigt verktyg att man kan göra mycket till websidor under byggnads prosses. Till exempel man kan färga dekorera planera layouten lägga till animationer och mycket mera. Med de två teknologier man kan konstrukturera websidans eller appens synlighet fast webbsidan blir inte dynamiskt. Att kunna göra webbsidan dynamiskt behöver vi JavaScript. JavaScript är ett programmeringsspråk och med hjälp av den kan vi göra webbsidan dynamiskt vilket betyder att vi kan ändra saker I webbsidan vi kan kontrollera information trafiken mellan webben och användaren. Man kan göra ändringar i html och CSS koderna lägga sidan 2d och 3d animationer video, man kan bygga formulär som användaren kan lämna sina uppgifter som personnummer, adress, namn, ålder mm. Det är JavaScript som möjliggör att användaren kan handla I webben, betala sina räkningar kommunicera med sin bank surfa I sociala medier och mycket mera. Allt detta betyder att JavaScript är ett dominerande programmering språk I webbens värld. Allt detta är kanske lotter komplicerat och krångligt för en i din ålder. Låt mig förklara på annat set.
