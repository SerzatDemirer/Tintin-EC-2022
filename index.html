Jul tid med familj och Tintin,(Del - 1 -)

Tintin frågade mig om mitt nya jobb och Jag berättade till han att nyligen började jag jobba i ett it-bolag, som front-end utvecklare. Då frågade han att vad gör en frontend utvecklare? Jag har berättat till han att front-end utvecklare bygger och underhåller webbsidor och appar liksom vi använder i våra mobiler och datorer idag. Då frågade Tintin; hur bygger ni webbsidor? Jag försökte förklara till honom;

Det är naturligtvis inte lätt att förklara detta, men jag ska försöka förklara det för dig så enkelt jag kan. I byggprocessen används tre grunda teknologier. De är HTML, CSS, och JavaScript.

HTML är förkortning av “Hyper Text Markup Language.” Även om det kallas "language" på engelska är html inte ett programmeringsspråk. Html är ett märkspråk och utgör strukturen för webbsidan. Html innehåller form av märk och styckesindelnings att skilja visst innehåll från varandra.
CSS är förkortning av “Cascading Style Sheet.” som namnet också avslöjar det är som CSS stylar webbsidorna. CSS är ett kraftigt verktyg att man kan göra mycket till websidor under byggprocess. Till exempel man kan färga dekorera planera layouten lägga till animationer och mycket mera.

Med html och CSS teknologier man kan konstrukturera webbsidans synlighet fast webbsidan blir inte dynamiskt. Att kunna göra webbsidan dynamiskt behöver vi JavaScript. JavaScript är ett programmeringsspråk och med hjälp av den kan vi göra webbsidan dynamiskt. Det betyder att vi kan ändra saker i webbsidan vi kan kontrollera information trafiken mellan servern och användaren. Man kan göra ändringar i html och CSS koderna och kan lägga till sidan 2d och 3d animationer, video, och man kan bygga formulär som användaren kan lämna sina uppgifter som personnummer, adress, namn, ålder mm. Det är JavaScript som möjliggör att användaren kan handla i webben, betala sina räkningar kommunicera med sin bank surfa i sociala medier och mycket mera. Med allt detta kraft blir JavaScript ett dominerande programmering språk i digitala världen.

Som exempel har jag skrivit två stycken kod i JavaScript att kunna visa saker som man kan göra med JavaScript. Ett hälsnings text för Jul och nytt år som animerar varje gång man förnyar sidan och en knap som byter färg när man trycker på.

En SPA (singel-page-applikation) är en webbapplikation implementering som bara laddar ett enda webbdokument till sidan och sedan uppdaterar dokumentet med hjälp av JavaScript API `er som XMLHttpRequest och Fetch. Singel-Page applikation kallas också client-side applikation. Populära företager som Facebook, Gmail, Netflix, Airbnb och PayPal är exemplar som använder singel-page applikation i sina webbsidor.
API (Applikation Program Interface) är ett protokoll som hjälper att program kommunicerar varandra via servern. Ajax stor för Asynchronous JavaScript And XML och gör http förfrågningar att skicka och ta tillbaka data från servern. JavaScript är programmering språk och har populära ramverk som React, Angular och Vue.JS. Ramverk är en viktig del av modern FRONT-END webbutveckling och ger möjlighet till utvecklare att bygga skalbar och interaktiva webbapplikationer.

Vad är OOP Objektorienterad programmen?

Objektorienterad programmering definieras som ett programmeringsparadigm. Det är inte ett specifikt språk utan byggt kring konceptet objekt. OOP byggs kring klasser och objekt. Den organiserar återanvändbar kodsnuttar eller klasser används sedan och återanvänds att skapa nya objekt med samma funktioner. APL, C++, C#, Java, Objektive-C och Swift är exempel på objektorienterad programmering. JavaScript är ett dynamiskt script språk som stöder prototypbaserad objektkonstruktion.
